# Hikari Biblioteca - Sistema WEB

Este é o projeto inicial da estrutura visual de um sistema WEB para gerenciamento de biblioteca, desenvolvido como Trabalho Prático 1 para a disciplina de Programação WEB II.

# Objetivos do Projeto:
Desenvolver a interface visual utilizando conceitos fundamentais de desenvolvimento WEB:
- Estruturação com HTML5.
- Estilização moderna com CSS (incluindo Flexbox e transições).
- Padrão arquitetural MVC (Model-View-Controller) usando ASP.NET Core.
- Navegação entre páginas e passagem de dados via Controllers.
